**Пользовательские команды** и **команды сообщения** теперь live! Эти команды появляются в контекстных меню для пользователей и сообщений, и в будущем они появятся еще больше.

## Команды пользователя

**Пользовательские команды** — это команды приложений, которые появляются в контекстном меню (правый клик или нажатие) пользователей. Это отличный способ поразить быстрые действия для вашего приложения, целевые пользователи.

```typescript title="src/app.commands.ts"
import { Injectable } from '@nestjs/common';
import { Context, UserCommand, UserCommandContext, TargetUser } from '@globalart/nestcord';
import { User } from 'discord. s';

@Injectable()
export class AppCommands {
    @UserCommand({ name: 'Get avatar' })
    public async getUserAvatar(
        @Context() [interaction]: UserCommandContext,
        @TargetUser() пользователь: User
    ) {
        return interaction. eply({
            embeds: [
                new MessageEmbed()
                    . etTitle(`аватар ${user.username}`)
                    . etImage(пользователь). isplayAvatarURL({ size: 4096, dynamic: true }))
            ]
        });
    }
}
```

## Команды сообщений

**Команды сообщений** — это команды, которые появляются в контекстном меню (правый клик или нажатие) сообщений. Это отличный способ поразить быстрые действия для вашего приложения, которые целевые сообщения.

```typescript title="src/app.commands.ts"
import { Injectable } from '@nestjs/common';
import { Context, MessageCommand, MessageCommandContext, TargetMessage } from '@globalart/nestcord';
import { Message } from 'discord. s';

@Injectable()
export class AppCommands {
    @MessageCommand({ name: 'Copy Message' })
    public async copyMessage(
        @Context() [interaction]: MessageCommandContext,
        @TargetMessage() message: Сообщение
    ) {
        return interaction. eply({ content: message.content });
    }
}
```
