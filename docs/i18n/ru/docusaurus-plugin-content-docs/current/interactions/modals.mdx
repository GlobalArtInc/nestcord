```typescript title="src/app.modals.ts"
import { Injectable } from '@nestjs/common';
import { Context, Modal, ModalContext } from '@globalart/nestcord';

@Injectable()
export class AppModals {
    @Modal('pizza')
    public onModal(@Ctx() [interaction]: ModalContext) {
      return interaction.reply({
        content: `Ваша любимая пицца : ${interaction.fields.getTextInputValue('pizza')}`
      });
    }
}
```

Чтобы получить динамический модальное окно
```typescript title="src/app.modals.ts"
import { Injectable } from '@nestjs/common';
import { Context, Modal, ModalContext,ModalParam } from '@globalart/nestcord';

@Injectable()
export class AppModals {
    @Modal('pizza/:value')
    public onModal(@Ctx() [interaction]: ModalContext, @ModalParam('value') value: string) {
      return interaction.reply({
        content: `Ваша любимая пицца ${value} : ${interaction.fields.getTextInputValue('pizza')}`
      });
    }
}
```

### Динамический модальный диалог
Вы можете создавать модальные диалоги с динамическим полем `id`. Это полезно для передачи метаданных в модальные диалоги.

Для создания динамического модального окна
```typescript
new ModalBuilder()
  .setTitle('What your fav pizza?')
  .setCustomId('pizza/12345')
  .setComponents([
    new ActionRowBuilder<ModalActionRowComponentBuilder>().addComponents([
      new TextInputBuilder()
        .setCustomId('pizza')
        .setLabel('???')
        .setStyle(TextInputStyle.Paragraph)
    ])
  ])
```

С модалями вы можете создавать всплывающие формы, которые позволяют пользователям предоставлять вам отформатированные данные через отправку. Мы рассмотрим, как создавать, отображать и получать формы модальных окон с помощью nestcord

Динамические модали используют формат [path-to-regexp](https://www.npmjs.com/package/path-to-regexp) для выполнения сопоставления с регулярными выражениями. Преобразуют строку пути, такую как `user/:name`, в регулярное выражение.
