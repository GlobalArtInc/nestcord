С модулями вы можете создавать всплывающие формы, которые позволяют пользователям предоставлять вам форматированные входы через отправку. Мы рассмотрим как создавать, показывать и получать формы модали, используя nestcord

```typescript title="src/app.modals.ts"
import { Injectable } from '@nestjs/common';
import { Context, Modal, ModalContext } from '@globalart/nestcord';

@Injectable()
export class AppModals {
    @Modal('pizza')
    public onModal(@Ctx() [interaction]: ModalContext) {
      return interaction. eply({
        content: `Your fav pizza : ${interaction.fields. etTextInputValue('pizza')}`
      });
    }
}
```

### Динамическая модаль

Вы можете создать модули с помощью динамического поля `id`. Это полезно для передачи метаданных модам.

Динамические модули используют формат [path-to-regexp](https://www.npmjs.com/package/path-to-regexp) для выполнения регулярного сопоставления. Превратите строку пути, такую как `user/:name` в регулярное выражение.

Создавать динамический модаль

```typescript
new ModalBuilder()
  .setTitle('What your fav pizza?')
  . etCustomId('pizza/12345')
  .setComponents([
    new ActionRowBuilder<ModalActionRowComponentBuilder>(). ddComponents([
      new TextInputBuilder()
        . etCustomId('pizza')
        . etLabel('???')
        .setStyle(TextInputStyle.Paragraph)
    ])
])
```

Получить динамический модаль

```typescript title="src/app.modals.ts"
импортировать { Injectable } из '@nestjs/common';
импортировать { Context, Modal, ModalContext,ModalParam } из '@globalart/nestcord';

@Injectable()
export class AppModals {
    @Modal('pizza/:value')
    public onModal(@Ctx() [interaction]: ModalContext, @ModalParam('value') значение: string) {
      возвращает взаимодействие. eply({
        content: `Your fav pizza ${value} : ${interaction.fields. etTextInputValue('pizza')}`
      });
    }
}
```
