## Кнопка

### Динамическая кнопка
Вы можете создавать кнопки с динамическим полем `id`. Это полезно для передачи метаданных кнопкам.

## Меню выбора

### Выбор роли
**Выбор роли** — это меню выбора для выбора ролей

Для создания Role Select:
```typescript
new RoleSelectMenuBuilder()
  .setCustomId('ROLE_SELECT_MENU')
  .setPlaceholder('Выберите роль')
  .setMaxValues(1)
  .setMinValues(1)
```

### Выбор канала
**Выбор канала** — это меню выбора для выбора каналов

### Упоминаемое выбрать
**Упоминаемое выбрать** - это меню выбора для выбора упоминаемых объектов (пользователей и ролей)

Существует несколько различных типов компонентов; эта документация описывает основы этой новой платформы и каждого примера.

Для получения String Select:
```typescript
import { Injectable } from '@nestjs/common';
import { Context, StringSelect, StringSelectContext, SelectedStrings } from '@globalart/nestcord';

@Injectable()
export class AppComponents {
    @StringSelect('STRING_SELECT_MENU')
    public onStringSelect(
      @Context() [interaction]: StringSelectContext,
      @SelectedStrings() selected: string[]
    ) {
      return interaction.reply({
        content: `Ваш выбранный цвет - ${selected.join(' ')}`
      });
    }
}
```

**Компоненты сообщений** — далее мы будем называть их «компонентами» — это фреймворк для добавления интерактивных элементов в сообщения, отправляемые вашим приложением или ботом. Они доступны, настраиваемы и просты в использовании.

Создать динамическую кнопку
```typescript
new ButtonBuilder()
  .setCustomId('click/12345')
  .setLabel('LABEL')
  .setStyle(ButtonStyle.Primary)
```

**Меню выбора** — еще один интерактивный компонент, отображаемый на сообщениях. На настольных компьютерах щелчок по меню выбора открывает раскрывающееся меню; на мобильных устройствах касание меню выбора открывает полусдвигающуюся панель с вариантами.

### Выбор строки
**Выбор строки** — это меню выбора для выбора предопределённых текстовых вариантов

```typescript title="src/app.components.ts"
import { Injectable } from '@nestjs/common';
import { Context, Button, ButtonContext } from '@globalart/nestcord';

@Injectable()
export class AppComponents {
    @Button('BUTTON')
    public onButton(@Context() [interaction]: ButtonContext) {
        return interaction.reply({ content: 'Кнопка нажата!' });
    }
}
```

Чтобы получить Dynamic button
```typescript
import { Injectable } from '@nestjs/common';
import { Context, Button, ButtonContext } from '@globalart/nestcord';

@Injectable()
export class AppComponents {
  @Button('click/:value')
  public onButton(
    @Context() [interaction]: ButtonContext,
    @ComponentParam('value') value: string
  ) {
    return interaction.reply({ content: `Button clicked! Value: ${value}` });
  }
}

```

Для создания Channel Select:
```typescript
new ChannelSelectMenuBuilder()
  .setCustomId('CHANNEL_SELECT_MENU')
  .setPlaceholder('Выберите канал')
  .setMaxValues(1)
  .setMinValues(1)
```

### Динамическое меню выбора
Все типы меню выбора поддерживают динамический `id` в формате [path-to-regexp](https://www.npmjs.com/package/path-to-regexp).

Для получения динамического меню выбора
```typescript
import { Injectable } from '@nestjs/common';
import { Context, StringSelect, StringSelectContext,SelectedStrings } from '@globalart/nestcord';

@Injectable()
export class AppComponents {
    @StringSelect('preferences/:item')
	public onPreferenceSelect(
		@Context() [interaction]: StringSelectContext,
		@SelectedStrings() values: string[],
		@ComponentParam('item') item: string
	) {
		return interaction.reply({
			content: `
      ${item} = ${values.join(',')}\n
      `
		});
	}
}

```
Динамические меню выбора совместимы с :
- `StringSelect`
- `UserSelect`
- `RoleSelect`
- `MentionableSelect`
- `ChannelSelect`

Для получения User Select:
```typescript
import { Injectable } from '@nestjs/common';
import { 
  Context,
  UserSelect,
  UserSelectContext,
  SelectedUser,
  ISelectedUser,
  SelectedMembers,
  ISelectedMembers
} from '@globalart/nestcord';

@Injectable()
export class AppComponents {
    @UserSelect('USER_SELECT_MENU')
    public onUserSelect(
      @Context() [interaction]: UserSelectContext,
      @SelectedUsers() users: ISelectedUsers, // Коллекция пользователей
      @SelectedMembers() members: ISelectedMembers // Коллекция участников
    ) {
      interaction.reply({
        content: `
        Выбранные пользователи - ${users.map(user => user.username).join(',')}\n
        Выбранные участники - ${members.map(member => member.user?.username).join(',')}
        `
      });
    }
}
```

Для получения выбора роли:
```typescript
import { Injectable } from '@nestjs/common';
import { 
  Context,
  RoleSelect,
  RoleSelectContext,
  SelectedRole,
  ISelectedRole,
} from '@globalart/nestcord';

@Injectable()
export class AppComponents {
    @RoleSelect('ROLE_SELECT_MENU')
    public onRoleSelect(
      @Context() [interaction]: RoleSelectContext,
      @SelectedRoles() roles: ISelectedRoles, // Коллекция ролей
    ) {
      interaction.reply({
        content: `
        Выбранные роли - ${roles.map(role => role.id).join(',')}
        `
      });
    }
}
```

Чтобы получить Channel Select:
```typescript
import { Injectable } from '@nestjs/common';
import { 
  Context,
  ChannelSelect,
  ChannelSelectContext,
  SelectedChannel,
  ISelectedChannel,
} from '@globalart/nestcord';

@Injectable()
export class AppComponents {
    @ChannelSelect('CHANNEL_SELECT_MENU')
    public onChannelSelect(
      @Context() [interaction]: ChannelSelectContext,
      @SelectedChannels() channels: ISelectedChannels, // Коллекция каналов
    ) {
      interaction.reply({
        content: `
        Выбранные каналы - ${channels.map(channel => channel.id).join(',')}
        `
      });
    }
}
```

Для создания динамического меню выбора:
```typescript
new StringSelectMenuBuilder()
  .setCustomId('preferences/color')
  .setPlaceholder('Выберите цвет')
  .setMaxValues(1)
  .setMinValues(1)
  .setOptions([
    { label: 'Красный', value: 'Red' },
    { label: 'Синий', value: 'Blue' },
    { label: 'Жёлтый', value: 'Yellow' }
  ])
```

**Кнопки** — это интерактивные компоненты, отображаемые на сообщениях. Пользователи могут нажимать на них, и при нажатии они отправляют [взаимодействие](https://discord.com/developers/docs/interactions/receiving-and-responding#interaction-object) в ваше приложение.

Динамические кнопки используют формат [path-to-regexp](https://www.npmjs.com/package/path-to-regexp) для выполнения сопоставления регулярных выражений. Преобразуют строку пути, такую как `user/:name`, в регулярное выражение.

Discord представил новые компоненты меню выбора:
- `String Select` (Строковое меню выбора)
- `User Select` (Меню выбора пользователя)
- `Role Select` (Меню выбора роли)
- `Mentionable Select` (Меню выбора упоминаемого элемента)
- `Channel Select` (Меню выбора канала)

Для создания String Select:
```typescript
new StringSelectMenuBuilder()
  .setCustomId('STRING_SELECT_MENU') // замените на ваш customId
  .setPlaceholder('Выберите ваш цвет')
  .setMaxValues(1)
  .setMinValues(1)
  .setOptions([
    { label: 'Красный', value: 'Red' },
    { label: 'Синий', value: 'Blue' },
    { label: 'Жёлтый', value: 'Yellow' }
  ])
```

### Выбор пользователя
**Выбор пользователя** — это меню выбора для выбора пользователей

Для создания User Select:
```typescript
new UserSelectMenuBuilder()
  .setCustomId('USER_SELECT_MENU')
  .setPlaceholder('Выберите пользователя')
  .setMaxValues(1)
  .setMinValues(1)
```

Чтобы создать Mentionable Select:
```typescript
new MentionableSelectMenuBuilder()
  .setCustomId('MENTIONABLE_SELECT_MENU')
  .setPlaceholder('Выберите пользователя/роль')
  .setMaxValues(1)
  .setMinValues(1)
```

Для получения Mentionable Select:
```typescript
import { Injectable } from '@nestjs/common';
import { 
  Context,
  MentionableSelect,
  MentionableSelectContext,
  SelectedUsers,
  ISelectedUsers,
  SelectedMembers,
  ISelectedMembers,
  SelectedRoles,
  ISelectedRoles,
} from '@globalart/nestcord';

@Injectable()
export class AppComponents {
    @MentionableSelect('MENTIONABLE_SELECT_MENU')
    public onMentionableSelect(
      @Context() [interaction]: MentionableSelectContext,
      @SelectedUsers() users: ISelectedUsers,
      @SelectedMembers() members: ISelectedMembers,
      @SelectedRoles() roles: ISelectedRoles
    ) {
      return interaction.reply({
        content: `
        Выбранные пользователи - ${users.map(user => user.username).join(',')}\n
        Выбранные участники - ${members.map(member => member.user?.username).join(',')}\n
        Выбранные роли - ${roles.map(role => role.name).join(',')}
        `
      });
    }
}
```
