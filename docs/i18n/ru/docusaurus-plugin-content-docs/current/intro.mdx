## Установка

**Требуется Node.js версии 16.6.0 или выше.**

```bash npm2yarn
$ yarn add @globalart/nestcord discord.js
```

Всякий раз, когда вам нужно обрабатывать данные событий, используйте декоратор `Context`.

**Особенности**

Затем создайте файл `app.update.ts` и добавьте декораторы `On`/`Once` для обработки событий Discord API:

```typescript title="src/app.update.ts"
import { Injectable, Logger } from '@nestjs/common';
import { Context, On, Once, ContextOf } from '@globalart/nestcord';
import { Client } from 'discord.js';

@Injectable()
export class AppUpdate {
    private readonly logger = new Logger(AppUpdate.name);

    public constructor(private readonly client: Client) {}

    @Once('ready')
    public onReady(@Context() [client]: ContextOf<'ready'>) {
        this.logger.log(`Бот вошел как ${client.user.username}`);
    }

    @On('warn')
    public onWarn(@Context() [message]: ContextOf<'warn'>) {
        this.logger.warn(message);
    }
}
```

Этот пакет использует лучшее из мира NodeJS под капотом. [Discord.js](https://github.com/discordjs/discord.js) — это самая мощная библиотека для создания ботов, а [Nest.js](https://github.com/nestjs) — это прогрессивный фреймворк для создания хорошо архитектурированных приложений. Этот модуль предоставляет быстрый и простой способ создания Discord ботов и глубокую интеграцию с вашим приложением NestJS.

import { NestCordModule } from '@globalart/nestcord';
import { Module } from '@nestjs/common';
import { IntentsBitField } from 'discord.js';
import { AppUpdate } from './app.update';

@Module({
    imports: [
        NestCordModule.forRoot({
            token: 'DISCORD_BOT_TOKEN',
            intents: [IntentsBitField.Flags.Guilds]
        })
    ],
    providers: [AppUpdate]
})
export class AppModule {}
