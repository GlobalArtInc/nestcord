NestCord имеет сниппеты для доступа к клиенту и его свойствам в вашем приложении.
Вы можете вводить менеджеры и утилиты discord.js с помощью конструктора.

```typescript title="src/app.service.ts"
import { Injectable } from '@nestjs/common';
import { Client, ChannelManager, GuildManager, UserManager, ShardClientUtil, ClientVoiceManager, WebSocketManager, REST } from 'discord. s';

@Injectable()
export class AppService {
    public constructor(
        private readonly client: Клиент,
        частные каналы только для чтения: ChannelManager,
        частные читаемые гильдии: Менеджер гильдии,
        частных читаемых пользователей: UserManager,
        Частный шард для чтения: ShardClientUtil,
        приватный голос: КлиентVoiceManager,
        частное чтение нас: WebSocketManager,
        закрытые только для чтения: REST
    ) {}
}
```

| Класс (Тип должен быть включен) | Свойства клиента (Доступ к)      | Описание                |
| ------------------------------- | -------------------------------- | ----------------------- |
| `Клиент`                        | `клиент`                         | Discord.js клиент       |
| `ChannelManager`                | `client.channels`                | Каналы клиента          |
| `Менеджер гильдии`              | `client.guilds`                  | Гильдии клиента         |
| `UserManager`                   | `client.users`                   | Пользователи клиента    |
| `ShardClientUtil`               | `client.shard`                   | Утилиты осколков        |
| `КлиентVoiceManager`            | `client.voice`                   | Голосовой менеджер      |
| `WebSocketManager`              | `client.ws`                      | Менеджер Websocket      |
| `REST`                          | `client.rest`                    | Менеджер REST           |
