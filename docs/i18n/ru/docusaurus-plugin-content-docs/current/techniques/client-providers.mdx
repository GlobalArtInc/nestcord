NestCord имеет фрагменты кода для доступа к клиенту и его свойствам в вашем приложении.
Вы можете внедрять менеджеры и утилиты discord.js клиента, используя конструктор.

```typescript title="src/app.service.ts"
import { Injectable } from '@nestjs/common';
import { Client, ChannelManager, GuildManager, UserManager, ShardClientUtil, ClientVoiceManager, WebSocketManager, REST } from 'discord.js';

@Injectable()
export class AppService {
    public constructor(
        private readonly client: Client,
        private readonly channels: ChannelManager,
        private readonly guilds: GuildManager,
        private readonly users: UserManager,
        private readonly shard: ShardClientUtil,
        private readonly voice: ClientVoiceManager,
        private readonly ws: WebSocketManager,
        private readonly rest: REST
    ) {}
}
```

| Класс (тип для внедрения) | Свойство клиента (доступ к) | Описание        |
|-----------------------------|----------------------------------|--------------------|
| `Client`                    | `client`                         | Клиент Discord.js  |
| `ChannelManager`            | `client.channels`                | Каналы клиента |  
| `GuildManager`              | `client.guilds`                  | Серверы клиента   | 
| `UserManager`               | `client.users`                   | Пользователи клиента    |
| `ShardClientUtil`           | `client.shard`                   | Утилиты шардов   |
| `ClientVoiceManager`        | `client.voice`                   | Менеджер голоса      |
| `WebSocketManager`          | `client.ws`                      | Менеджер Websocket  |
| `REST`                      | `client.rest`                    | Менеджер REST       |
