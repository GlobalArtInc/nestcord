Теперь мы можем добавить несколько правил валидации в наш `CreateUserDto`. Мы делаем это, используя декораторы из пакета class-validator, подробно описанные [здесь](https://github.com/typestack/class-validator#validation-decorators). Таким образом, любой маршрут, который использует `CreateUserDto`, будет автоматически применять эти правила валидации.

```typescript title="src/users/users.commands.ts"
import { Injectable } from '@nestjs/common';
import { Context, SlashCommand, SlashCommandContext } from '@globalart/nestcord';

@Injectable()
export class UsersCommands {
    @SlashCommand({
        name: 'create',
        description: 'Создать нового пользователя'
    })
    public async onCreateUser(
        @Context() [interaction]: SlashCommandContext,
        @Options(new ValidationPipe({ validateCustomDecorators: true })) createUserDto: CreateUserDto
    ) {
        return interaction.reply({ content: `Пользователь создан: ${createUserDto.name}` });
    }
}
```

[Дополнительную информацию см. в разделе Фильтры исключений](https://docs.nestjs.com/exception-filters#exception-filters)
:::

```typescript title="src/users/dto/create-user.dto.ts"
import { IsNotEmpty, IsString, IsEmail } from 'class-validator';

export class CreateUserDto {
    @StringOption({
        name: 'name',
        description: 'Ваше имя',
        required: true
    })
    name: string;

    @IsEmail()
    @StringOption({
        name: 'text',
        description: 'Ваш адрес электронной почты',
        required: false
    })
    email: string;
}
```

`ValidationPipe` — это встроенный pipe, который можно использовать для проверки данных, поступающих от клиента. Под капотом он использует пакет [class-validator](https://github.com/typestack/class-validator). `ValidationPipe` предоставляет удобный подход к применению правил валидации для всех входящих полезных нагрузок клиента, где конкретные правила объявляются с помощью простых аннотаций в объявлениях DTO в каждом модуле.

```typescript title="src/decorators/validated-options.decorator.ts"
import { ValidationPipe, PipeTransform } from '@nestjs/common';

export const ValidatedOptions = (...dataOrPipes: PipeTransform[] | string) => {
    return Options(...dataOrPipes, new ValidationPipe());
};
```

:::tip
Вы также можете использовать `ValidationPipe` для преобразования и валидации полезной нагрузки запроса.

:::tip
Вы можете создавать фильтры для обработки ошибок валидации запросов и ответов.
