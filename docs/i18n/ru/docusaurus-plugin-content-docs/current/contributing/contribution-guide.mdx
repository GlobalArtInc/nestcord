# Вклад

Если вы хотите внести свой вклад в код nestcord или документацию, не стесняйтесь форкнуть репозиторий и отправить запрос на слияние
. Мы используем ESLint для обеспечения согласованного стиля кодирования, так что настройка в редакторе по вашему выбору
снижает трение в этом проекте.

## Настройка

Вам понадобится [Node.js](http://nodejs.org) **версия 16+** и [npm](https://www.npmjs.com/).

Чтобы быть готовыми к работе на базе кода, выполните следующие действия:

1. Форк и клонирование репозитория. Убедитесь, что у вас есть актуальная информация о **мастер** ветке!
2. Установите зависимости, используя `npm ci`
3. Напишите свое сердце!
4. Запустите `npm lint` для запуска ESLint и убедитесь, что изменения соответствуют нашему стилю
5. [Отправить Pull request](https://github.com/GlobalArtInc/nestcord/compare)

Высокий уровень обзора используемых инструментов:

- [TypeScript](https://www.typescriptlang.org/) как язык разработки
- [Eslint](https://eslint.org/) для код-стиля
- [Prettier](https://prettier.io/) для форматирования кода
