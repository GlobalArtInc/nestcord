# Способствование

I'm ready to translate from English (US) to Russian. Please provide the text you'd like me to translate, and I'll return only the translated version without any explanations or extra content.

Вам потребуется [Node.js](http://nodejs.org) **версии 16+** и [npm](https://www.npmjs.com/).

Чтобы подготовиться к работе с кодовой базой, пожалуйста, выполните следующее:

Высокоуровневый обзор используемых инструментов:

- [TypeScript](https://www.typescriptlang.org/) в качестве языка разработки
- [Eslint](https://eslint.org/) для стиля кода
- [Prettier](https://prettier.io/) для форматирования кода

Если вы хотите внести свой вклад в кодовую базу или документацию nestcord, не стесняйтесь форкировать репозиторий и отправить pull request. Мы используем ESLint для обеспечения единообразного стиля кодирования, поэтому его настройка в редакторе по вашему выбору упрощает процесс внесения вклада в этот проект.

1. Сделайте форк и клонируйте репозиторий. Убедитесь, что вы обновили **master** ветку!
2. Установите зависимости, используя `npm ci`
3. Кодируйте от всего сердца!
4. Запустите `npm lint`, чтобы запустить ESLint и убедиться, что ваши изменения соответствуют нашему руководству по стилю
5. [Отправьте pull request](https://github.com/GlobalArtInc/nestcord/compare)
