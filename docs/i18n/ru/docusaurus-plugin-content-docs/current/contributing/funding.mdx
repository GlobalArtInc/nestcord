# Финансирование

NestCord - это проект с открытым исходным кодом с его постоянной разработкой, который стал возможным благодаря поддержке сообщества.\
Эта библиотека является результатом длительной дороги, полной бессонных ночей, работы после часов и занятых выходных.

Если вы хотите поддержать NestCord финансово, спасибо!
