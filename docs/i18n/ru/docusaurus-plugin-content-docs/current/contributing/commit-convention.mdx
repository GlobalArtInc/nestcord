## Соглашение по формату сообщений коммитов Git

#### Краткое резюме:

Заголовки коммитов должны соответствовать следующему регулярному выражению:

I'm ready to translate text from English (US) to Russian. Please provide the text you would like me to translate, and I will return only the translated text without any explanations or additional content.

**Заголовок** обязателен, а **область действия** заголовка не обязательна.

Если префикс `feat`, `fix` или `perf`, он будет отображаться в журнале изменений. Однако если есть любое [BREAKING CHANGE](#footer), коммит всегда будет отображаться в журнале изменений.

I appreciate you sharing the heading, but I notice there's no actual text content to translate - just the word "Subject" as a header.

Could you please provide the English text you'd like me to translate to Russian? Once you share the content, I'll be happy to provide the translation.

i understand. i will:
- use imperative present tense (base form of verbs)
- not capitalize the first letter
- not end with a period

ready to translate when you provide the text

Нижний колонтитул должен содержать любую информацию об **изменениях, нарушающих совместимость**, а также здесь следует ссылаться на проблемы GitHub, которые закрывает этот коммит.

Другие префиксы остаются на ваше усмотрение. Предлагаемые префиксы — `docs`, `chore`, `style`, `refactor` и `test` для задач, не связанных с журналом изменений.

```js
/^(revert: )?(feat|fix|docs|style|refactor|perf|test|workflow|build|ci|chore|types|wip)(\(.+\))?: .{1,72}/;
```

Это адаптировано из [соглашения о коммитах Angular](https://github.com/conventional-changelog/conventional-changelog/tree/master/packages/conventional-changelog-angular).

Если коммит отменяет предыдущий коммит, он должен начинаться с `revert:`, за которым следует заголовок отменяемого коммита. В теле сообщения должно быть написано: `This reverts commit <hash>.`, где hash — это SHA отменяемого коммита.

Как в **теме письма**, используйте повелительное наклонение, настоящее время.
В теле письма должны быть указаны мотивация к изменению и отличие от предыдущего поведения.
