Интерфейс ServiceOption:
| Свойство                        | Тип                                        | Описание                                                                                              |
|---------------------------------|--------------------------------------------|-------------------------------------------------------------------------------------------------------|
| `name`                          | `string`                                   | Имя сервиса                                                                                          |
| `url`                           | `string`                                   | URL API для отправки статистики                                                                      |
| `proxy`                         | `AxiosProxyConfig` или `undefined`         | Конфигурация прокси                                                                                   |
| `bodyData`                      | `object`                                   | Данные тела запроса                                                                                   |
| `headerData`                    | `object`                                   | Данные заголовков                                                                                     |
| `schedule`                      | `StatCronExpression` или `string`          | Выражение Crontab                                                                                     |

# Использование

Поздравляем! Вы успешно зарегистрировали cronjob в NestCord!

### Глобальные параметры
Интерфейс NestCordStatReporterModule:
| Свойство                        | Тип                                        | Описание                                                                                              |
|---------------------------------|--------------------------------------------|-------------------------------------------------------------------------------------------------------|
| `services`                      | `ServiceOption[]`                          | Сервисы                                                                                               |
| `proxy`                         | `AxiosProxyConfig` или `undefined`         | Конфигурация прокси                                                                                   |
| `development`                   | `boolean`                                  | Если true, модуль будет отключён                                                                     |
| `log`                           | `boolean`                                  | Отправлять журнал при отправке статистики                                                            |

После завершения процесса установки мы можем импортировать `NestCordStatReporterModule` вместе с вашим `NestCordModule` в корневой `AppModule`:
```typescript
import { Module } from '@nestjs/common';
import { AppService } from './app.service';
import { NestCordModule, 
         NestCordStatReporterModule, 
         StatCronExpression, 
         NestCordLocalizationModule, 
         DefaultLocalizationAdapter, 
         UserResolver 
} from '@globalart/nestcord';

@Module({
    imports: [
      NestCordModule.forRoot({
        token: process.env.DISCORD_TOKEN,
          intents: [
          IntentsBitField.Flags.Guilds,
          IntentsBitField.Flags.DirectMessages,
          IntentsBitField.Flags.GuildMembers,
          IntentsBitField.Flags.GuildMessages,
          IntentsBitField.Flags.MessageContent
        ],
        prefix: '!',
        development: [process.env.DISCORD_TEST_GUILD]
      }),
      NestCordStatReporterModule.forRoot({
        services: [
          {
            name: 'top.gg',
            actions: [
              {
                name: 'название действия',
                url: 'https://top.gg/bots/:bot_id/stats',
                bodyData: { server_count: '{{serverCount}}', shard_count: '{{shardCount}}' },
              }
            ],
            headerData: { Authorization: process.env.TOP_GG_TOKEN },
            schedule: StatCronExpression.EVERY_5_MINUTES // или вы можете использовать выражение crontab как */1 * * * * ,
          },
        ],
      }),
    ],
    providers: [AppService]
})
export class AppModule {
}
```
В `services` должны быть перечислены все сервисы, на которых отслеживается бот, и должно быть передано тело запроса. `{{serverCount}}` и `{{shardCount}}` будут автоматически заменены.

Это легковесный модуль для отчетности статистики NestCord. Этот модуль отправляет данные с вашего бота в различные сервисы мониторинга ботов.
