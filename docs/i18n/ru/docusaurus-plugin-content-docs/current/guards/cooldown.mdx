Декоратор `@Cooldown()` устанавливает ограничение частоты запросов для каждого пользователя на любую команду или обработчик компонента. Он автоматически применяет `CooldownGuard` — необходимость в `@UseGuards()` отсутствует.

Импортируйте `NestCordCooldownModule` один раз в ваш корневой модуль. Выберите сервер хранилища, который соответствует вашему сценарию использования:

Сохраняет кулдауны в Redis. Переживает перезагрузки и используется совместно всеми шардами и процессами. Требует [`ioredis`](https://github.com/redis/ioredis):

```typescript
NestCordCooldownModule.forRoot({
  storage: { type: 'redis', options: { url: 'redis://localhost:6379' } },
})
```

```typescript
@Permissions(PermissionFlagsBits.BanMembers)
@Cooldown(10_000)
@UserCommand({ name: 'Забанить пользователя' })
async onBan(@Context() [interaction]: UserCommandContext) { ... }
```

## Обработка исключения

Зарегистрируйте фильтр на обработчике, классе или глобально:

```typescript
// На уровне класса
@UseFilters(CooldownExceptionFilter)
@Injectable()
export class AppCommands { ... }

// Глобально (main.ts)
app.useGlobalFilters(new CooldownExceptionFilter());
```

Сохраняет периоды восстановления в JSON-файл на диске. Переживает перезагрузки. Подходит для однопроцессных ботов, которые не могут использовать Redis.

Когда пользователь находится на перезарядке, `CooldownGuard` выбрасывает `CooldownException`, который раскрывает `remainingMs` — точное оставшееся время до истечения перезарядки.

:::tip
При использовании **Redis** бэкенда `remainingMs` получается из сохранённой временной метки, а не из TTL Redis, поэтому остаётся точным даже при небольшом смещении часов между перезагрузками.
:::

:::info Модуль NestCordCooldownModule не импортирован
Если вы используете `@Cooldown()` без импорта `NestCordCooldownModule`, guard автоматически переключится на хранилище в памяти. Это удобно для быстрой настройки, но означает, что таймауты сбросятся при перезагрузке.
:::

```typescript title="src/filters/cooldown.filter.ts"
import { ArgumentsHost, Catch, ExceptionFilter } from '@nestjs/common';
import { NestCordArgumentsHost, CooldownException, AnyContext } from '@globalart/nestcord';

@Catch(CooldownException)
export class CooldownExceptionFilter implements ExceptionFilter {
  catch(exception: CooldownException, host: ArgumentsHost) {
    const [interaction] = NestCordArgumentsHost.create(host).getContext<AnyContext>();

    const seconds = (exception.remainingMs / 1000).toFixed(1);
    return interaction.reply({
      content: `⏳ Вы на кулдауне. Попробуйте снова через **${seconds}s**.`,
      ephemeral: true,
    });
  }
}
```

```typescript title="src/app.commands.ts"
import { Injectable } from '@nestjs/common';
import { Context, Cooldown, SlashCommand, SlashCommandContext } from '@globalart/nestcord';

@Injectable()
export class AppCommands {
  @Cooldown(5_000) // 5 секунд
  @SlashCommand({ name: 'roll', description: 'Бросить случайное число' })
  async onRoll(@Context() [interaction]: SlashCommandContext) {
    const result = Math.floor(Math.random() * 100) + 1;
    return interaction.reply({ content: `🎲 Вы выбросили **${result}**!` });
  }
}
```

Примените `@Cooldown(ms)` к любому обработчику. Аргумент — это длительность перезагрузки в **миллисекундах**.

```typescript
@Cooldown(86_400_000) // 24 часа
@SlashCommand({ name: 'daily', description: 'Получить ежедневное вознаграждение' })
async onDaily(@Context() [interaction]: SlashCommandContext) {
  return interaction.reply({ content: '🎁 Ежедневное вознаграждение получено!' });
}
```
