:::caution

Текстовая команда зависит от содержания сообщения, но к сожалению, Discord планирует удалить содержимое сообщений для верифицированных ботов и
приложений, у которых 100 или более серверов. Следовательно, вы не можете использовать текстовые команды, если ваш бот не может получить доступ к содержимому сообщений.

[Прочитать Discord сообщение](https://support-dev.discord.com/hc/en-us/articles/4404772028055-Message-Content-Access-Deprecation-for-Verified-Bots)

:::

Создайте простой обработчик команд для сообщений с помощью `@TextCommand`.

```typescript title="src/app.service.ts"
импортировать { Injectable } из '@nestjs/common';
импортировать { Context, TextCommandContext, Arguments } из '@globalart/nestcord';

@Injectable()
export class AppService {
    @TextCommand({
        name: 'ping',
        описание: 'Команда пинга! ,
    })
    public onPing(@Context() [message]: TextCommandContext) {
        return message. eply('pong!');
    }
}
```

## Аргументы

Вы также можете использовать аргументы с текстовыми командами. Аргументы - это слова после названия команды.

```typescript title="src/app.service.ts"
импортировать { Injectable } из '@nestjs/common';
импортировать { Context, TextCommandContext, Arguments } из '@globalart/nestcord';

@Injectable()
export class AppService {
    @TextCommand({
        name: 'echo',
        описание: 'Команда Echo! ,
    })
    public onEcho(@Context() [message]: TextCommandContext, @Arguments() args: string[]) {
        return message. eply(args.join(' '));
    }
}
```
