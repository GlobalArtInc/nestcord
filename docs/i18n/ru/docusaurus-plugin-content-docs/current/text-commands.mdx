Вы также можете использовать аргументы с текстовыми командами. Аргументы — это слова после имени команды.

Текстовая команда зависит от содержания сообщения, но, к сожалению, Discord планирует удалить доступ к содержанию сообщений для проверенных ботов и приложений, которые используются на 100 и более серверах. Таким образом, вы не сможете использовать текстовые команды, если ваш бот не может получить доступ к содержанию сообщений.

```typescript title="src/app.service.ts"
import { Injectable } from '@nestjs/common';
import { Context, TextCommand, TextCommandContext } from '@globalart/nestcord';

@Injectable()
export class AppService {
    @TextCommand({
        name: 'hello',
        description: 'Скажи привет',
        aliases: ['hi', 'hey'],
    })
    public onHello(@Context() [message]: TextCommandContext) {
        return message.reply('Привет!');
    }
}
```

# Простой обработчик команд для сообщений с помощью `@TextCommand`

```python
from discord.ext import commands
import discord

class SimpleCommandHandler(commands.Cog):
    def __init__(self, bot):
        self.bot = bot

    @commands.command(name='hello')
    async def hello_command(self, ctx):
        """Простая команда приветствия"""
        await ctx.send(f'Привет, {ctx.author.name}!')

    @commands.command(name='ping')
    async def ping_command(self, ctx):
        """Проверка задержки бота"""
        await ctx.send(f'Понг! {round(self.bot.latency * 1000)}мс')

    @commands.command(name='info')
    async def info_command(self, ctx, *, user: discord.User = None):
        """Получить информацию о пользователе"""
        user = user or ctx.author
        embed = discord.Embed(title=f'Информация о {user.name}')
        embed.add_field(name='ID', value=user.id)
        embed.add_field(name='Создано', value=user.created_at.strftime('%d.%m.%Y'))
        await ctx.send(embed=embed)

async def setup(bot):
    await bot.add_cog(SimpleCommandHandler(bot))
```

**Основные особенности:**
- `@commands.command()` — декоратор для определения команды
- `ctx` — контекст команды (информация о сообщении, авторе и т.д.)
- `await ctx.send()` — отправка ответа
- Поддержка параметров команды (`user`, `*args`, `**kwargs`)
